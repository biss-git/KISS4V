【はじめに】

KISS4V（以降では"本アプリ"と表記）はボイスロイドの読み上げをサポートするアプリです。
本アプリはWindowsのバックグランドで動作します。

chrome拡張機能のKISS4V_chrome拡張 ( https://chrome.google.com/webstore/detail/kiss4vchrome%E6%8B%A1%E5%BC%B5/ebobjljncahboghpchcidlkdcdhkhbdp?hl=ja )
などと組み合わせて利用することでボイスロイドでの読み上げの自動化が可能です。


【動作環境】

Windows 10 64bit版で動作を確認しています。
また、.NET Framework 4.7.2 以上が必要です。

対応するボイスロイド製品はVoiceroidUtil ( https://www.nicovideo.jp/watch/sm28526328 )
と同じです。


【使い方】

本テキストと同じフォルダ内にある「KISS4V.exe」をダブルクリックして起動してください。
タスクトレイにアプリが追加されれば起動しています。

読み上げを自動化する場合はKISS4V_chrome拡張などのアプリと組み合わせてご利用ください。
